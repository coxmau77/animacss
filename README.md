# Animacss: Motor de Animación Nativa para Storyboard

**Animacss** es una exploración técnica que une la generación de imágenes por Inteligencia Artificial con el desarrollo frontend puro. El proyecto demuestra cómo transformar una secuencia estática (un _sprite sheet_ o storyboard) en una experiencia audiovisual fluida, infinita y optimizada para dispositivos móviles, utilizando exclusivamente el ecosistema web nativo.
